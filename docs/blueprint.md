# **App Name**: Monto Fácil

## Core Features:

- Registro de Pagos Diarios: Permite al usuario ingresar fácilmente el nombre del cliente y el monto cobrado para cada transacción.
- Total Diario Automático: Calcula y muestra automáticamente la suma total de todos los pagos registrados en el día actual en tiempo real.
- Compartir Resumen por WhatsApp: Genera un resumen del total diario y lo prepara para compartirlo de forma sencilla a través de WhatsApp.
- Exportar Reporte PDF: Crea y descarga un archivo PDF con el detalle de todos los pagos del día y el monto total cobrado.
- Restablecimiento Diario de Datos: Proporciona una opción para borrar los registros del día anterior y comenzar uno nuevo, manteniendo la pantalla limpia y lista.
- Almacenamiento Seguro de Datos: Persistencia de los registros de pago del día en una base de datos para asegurar su disponibilidad hasta el final del día y para la generación de reportes.

## Style Guidelines:

- Esquema de color claro que evoca profesionalismo y claridad, facilitando la visualización de los datos.
- Color primario: Azul tenue (#5E7DBA). Representa confiabilidad y eficiencia, sin ser intrusivo.
- Color de fondo: Gris-azul muy claro (#F0F1F4). Ofrece una base limpia y espaciosa que no distrae la vista.
- Color de acento: Cyan vibrante (#26A1D9). Utilizado para acciones clave y resaltar elementos importantes, proporcionando un contraste adecuado.
- La fuente 'Inter' (sans-serif) para todo el texto, garantizando alta legibilidad, un aspecto moderno y claro que es fácil de leer para montos y nombres, como se solicitó.
- Uso de íconos simples y directos, como un signo de más para añadir un nuevo registro, un botón de compartir para WhatsApp y un ícono de PDF para la exportación de informes.
- Diseño optimizado para móviles con campos de entrada grandes y claros, así como una visualización prominente del total diario para facilitar el uso en movimiento. Las dos columnas solicitadas serán 'Cliente' y 'Monto'.
- Animaciones sutiles y rápidas para confirmar la entrada de datos o la actualización del total, mejorando la experiencia del usuario sin distracciones.
